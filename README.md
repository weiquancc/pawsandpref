## How to Use

1. Open the application in your web browser
2. Wait for the cat images to load
3. Swipe right (or click the heart button) to like a cat
4. Swipe left (or click the X button) to pass on a cat
5. After viewing all 15 cats, see your summary of liked cats
6. Click "Start Over" to try again!

## Setup & Deployment

### Local Development

1. Clone this repository
2. Open `index.html` in a web browser
3. That's it! No build process required.

### GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select the main branch as the source
4. Your site will be available at `https://[username].github.io/pawsandpref/`
